Voter File

🛠 Data Manager

Upload a CSV to enrich existing voters with phone numbers, emails, and custom data. Matches by Voter ID first, then name + address.

Column Mapping

Preview (first 5 rows)

Upload the county's "who voted" list to mark voters as having voted. Matches by State File ID (voter ID), registration number, county file ID, or VANID. For primaries, choose which party ballot — voters get a red (R) or blue (D) tag.

CSV needs at least one ID column: StateFileID, VoterID, RegistrationNumber, CountyFileID, or VANID. If the CSV has a party column, it overrides the global party selection above.

Custom columns store extra data on voters (e.g. "voter_score_2024", "precinct_captain", etc.). These columns appear in CSV import and enrichment column mapping.
